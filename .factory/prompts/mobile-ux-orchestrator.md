# Mobile UX Orchestrator

Deploy 6 conversion-optimizer droids in parallel to perfect the mobile experience.

---

## Mission

Transform `/offer` into a **flawless mobile-first experience** for cold Facebook ad traffic.

## Context

- **Traffic**: 100% mobile from Facebook video ads
- **Current gap**: Hero section has no visual element — text only
- **Landing page**: `/src/app/offer/page.tsx` and `/src/components/offer/*`
- **Design system**: See `docs/03-sumi-breath.md` for visual doctrine

## Critical Requirements

1. Every droid analyzes the current mobile experience first
2. Propose visual/layout improvements — not copy changes
3. Output ready-to-ship code changes

## The 6 UX Levers

| #   | Lever              | Mission                                            |
| --- | ------------------ | -------------------------------------------------- |
| 1   | **Hero Visual**    | Add compelling visual element above the fold       |
| 2   | **Scroll Flow**    | Optimize content hierarchy for thumb scrolling     |
| 3   | **Touch Targets**  | Ensure all interactive elements are thumb-friendly |
| 4   | **Visual Proof**   | Make testimonials and trust signals scannable      |
| 5   | **Image Strategy** | Optimize product/lifestyle imagery for mobile      |
| 6   | **Performance**    | Ensure fast load times on 4G connections           |

## Success Criteria

The page loads fast, looks stunning, and feels native to mobile — someone scrolling in bed at midnight can navigate and buy without friction.
